### Set up Instrunction

- Install dependecies with `$ npm install`;
- Run migratuons with `$ npx prisma migrate dev` ;
- To run in dev mode use  `$ npm run dev`;
- To create prod build  `$ npm run build`;
- To run in  prod use  `$ npm start`
